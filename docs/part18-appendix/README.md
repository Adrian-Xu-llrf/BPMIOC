## 💡 使用建议

### 第一次使用

1. **浏览术语表** → 熟悉常用术语
2. **查看FAQ** → 了解常见陷阱
3. **收藏外部资源** → 建立学习书签
4. **打印快速参考** → 放在手边

### 日常使用

```
遇到问题
  ↓
查看FAQ (90%问题已覆盖)
  ↓
未解决？查看外部资源
  ↓
仍未解决？查看官方文档或提问
```

### 高效查找

**浏览器内搜索**:
```
Ctrl+F (Linux/Windows)
Cmd+F (Mac)
```

**grep搜索**:
```bash
# 在所有文档中搜索关键词
grep -r "dlopen" docs/

# 在FAQ中搜索错误信息
grep -i "cannot load" docs/part18-appendix/faq.md
```
