```markdown

- [ ] 运行第一个IOC

- 准备编译环境

### 17:30-18:30 | EPICS Base安装 (60分钟)
**学习内容**:
- Part 1: 03-epics-installation.md
- 章节: 2. EPICS Base编译

- EPICS_BASE路径设置
- 编译命令: make clean && make
- 测试: softIoc命令

### 18:30-19:30 | 运行测试 (60分钟)
**实验**:
- 运行softIoc
- 创建简单的.db文件
- 测试caget/caput

**完成情况**:
- [x] softIoc运行成功
- [x] 创建test.db
- [x] caget测试通过

### 19:30-20:00 | 记录和总结 (30分钟)
**今日收获**:
- 学会了: EPICS Base编译和安装
- 理解了: IOC的基本概念
- 实现了: 运行第一个IOC

**遇到的问题**:
1. readline库缺失
   - 解决: sudo apt-get install libreadline-dev
   - 结果: 编译成功

**明日计划**:
- 继续Part 1文档
- 编译BPMIOC
- 启用模拟模式
```

## 🎯 使用技巧

### 技巧1: 番茄工作法

```
25分钟专注学习
 ↓
5分钟休息
 ↓
重复4次
 ↓
15-30分钟长休息
```

### 技巧2: 时间块预留

```
每天预留30分钟"弹性时间"
用于:
- 处理突发问题
- 延长重要内容学习
- 额外的实践练习
```

### 技巧3: 定期回顾

```
每日: 睡前5分钟回顾今日学习
每周: 周日整理本周笔记
每月: 回顾整体进度
```
